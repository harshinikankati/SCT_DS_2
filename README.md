# 🚢 Titanic Dataset – Exploratory Data Analysis (Power BI)

## 📌 Internship Task 02 – Data Cleaning & EDA

This project is part of my **internship Task 02**, where I performed **data cleaning and exploratory data analysis (EDA)** on the Titanic dataset using **Power BI**.  
The objective of this task is to explore relationships between variables and identify patterns and trends affecting passenger survival.

---

## 🎯 Task Objectives
- Perform data cleaning using Power Query
- Handle missing and inconsistent data
- Create meaningful visualizations for EDA
- Analyze survival trends using Power BI dashboards

---

## 📂 Dataset Information
- Dataset: Titanic (Kaggle)
- File used: `train.csv`
- Total Records: 891 passengers

---

## 🧹 Data Cleaning Process
The following data cleaning steps were performed in **Power BI (Power Query)**:

- **Age**: Missing values replaced with average age
- **Embarked**: Missing values replaced with mode
- **Cabin**: Column removed due to excessive missing values
- Verified data types for categorical and numerical columns

---

## 📊 Exploratory Data Analysis (EDA)

### Dashboard Pages Created:

### 🟦 Page 1: Overview Dashboard
- Total Passengers
- Total Survivors
- Total Non-Survivors
- Survival Rate
- Survival Distribution Chart

---

### 🟩 Page 2: Demographics Analysis
- Survival by Gender
- Survival by Passenger Class
- Age Distribution (Binned)
- Family Size vs Survival

---

## 📈 Key Insights
- Female passengers had a higher survival rate compared to males
- Passengers in first class had better survival outcomes
- Children showed higher survival chances than adults
- Passengers traveling with small families survived more than those alone

---

## 🛠 Tools & Technologies Used
- Power BI
- Power Query
- DAX
- Microsoft Excel (dataset source)

## 📌 Conclusion
This exploratory data analysis shows that **gender, passenger class, age, and family size** played a significant role in survival outcomes on the Titanic.  
The project demonstrates how **Power BI can be effectively used for data cleaning, EDA, and storytelling through dashboards**.
